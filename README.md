# ToDo API на Go + Fiber + GORM

Простой REST API для управления задачами (ToDo), написанный на Go с использованием фреймворка Fiber и ORM GORM.

## 📚 Описание

Это учебный проект для изучения backend-разработки на Go. Приложение реализует базовый CRUD (Create, Read, Update, Delete) для задач.

## 🛠 Технологии

- **Go** — язык программирования
- **Fiber** — веб-фреймворк (аналог Express для Node.js)
- **GORM** — ORM для работы с базой данных
- **PostgreSQL** — реляционная база данных
- **Godotenv** — загрузка переменных окружения из .env файла
